# 🌊 Water Quality Analysis using Python

## 📖 Description

This project analyzes water quality data to identify trends and patterns in environmental parameters.
It uses Python libraries to clean data and generate visual insights such as BOD levels, dissolved oxygen, and correlations.

---

## 🚀 Features

* Data cleaning (missing values & duplicates)
* Exploratory Data Analysis (EDA)
* Trend analysis over years
* State-wise comparison
* Multiple visualizations (line, bar, histogram, heatmap, pie chart)

---

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📂 Files

* `project.py` → Main analysis code
* `dataset.csv` → Input dataset

---

## ⚙️ How to Run

```bash
git clone https://github.com/your-username/water-quality-analysis.git
cd water-quality-analysis
pip install pandas matplotlib seaborn
python project.py
```

---

## 📊 Output

The project generates:

* Trend graphs of BOD levels
* State-wise pollution comparison
* Distribution plots
* Correlation heatmap
* Water body type analysis
